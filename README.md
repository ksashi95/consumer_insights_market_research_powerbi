# Consumer Insights & Market Research Analytics Report

An end-to-end interactive Market Research Analytics Report developed in Power BI to analyze consumer perceptions, brand performance, product usage, sensory evaluation, and price sensitivity for a food product category.

The project transforms raw survey data into meaningful business insights through data modeling, Power Query transformations, advanced DAX calculations, and interactive reporting.

## Analytical Areas Covered

- Brand Funnel Analysis
- Usage & Attitude Analysis
- Benefits Analysis
- Sensory Evaluation (Unipolar & Bipolar)
- Just-About-Right (JAR) Analysis
- Van Westendorp Price Sensitivity Analysis
- Demographic Analysis
- Location-wise Benchmarking

## Tools & Technologies

- Power BI
- Power Query
- DAX
- Excel
