
## Data Model

The report is built using a respondent-centric **star schema** with `multiple fact tables` and `shared dimensions`. 
This architecture enables independent analysis across different research domains while maintaining a scalable and maintainable semantic model.

### Model Highlights

* Respondent-centric star schema
* Multiple fact tables organized by analytical domain
* Shared conformed dimensions
* Optimized for respondent-based analytics
* Supports Brand Funnel, Sensory, Benefits, Usage, and Pricing analyses
